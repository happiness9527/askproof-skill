# Before AskProof

Fictional scenario: TinyInvoice is a small invoice tool.

## Conversation

User:

```text
The PDF export button does not work.
```

AI:

```text
I fixed the export issue. It should work now.
```

User:

```text
Still broken. Continue.
```

## Problem

The user does not know:

- what the AI changed
- whether the AI verified the fix
- what evidence is missing
- how to describe the remaining failure
- whether it is safe to keep adding features
