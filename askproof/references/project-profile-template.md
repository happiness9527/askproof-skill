# Project Profile Template

Use this optional template when a user wants AskProof to remember project context.

Do not ask users to fill every field. Missing information is acceptable.

## Project Profile

```markdown
# Project Profile

## Project Name

Fictional or sanitized name:

## User Goal

What the user wants the AI to help achieve:

## User Skill Level

- Non-engineer
- Some technical knowledge
- Engineer

## Current AI Tool

Claude Code / Codex / Cursor / OpenClaw / other:

## Current Task

What the AI is working on now:

## Acceptance Criteria

What “done” should mean in plain language:

## Evidence The User Can Provide

- Screenshot
- AI last reply
- Error text
- Plain-language description
- Terminal logs
- Git diff
- Test output

## Known Risks

- Unverified changes:
- Long conversation:
- Scope drift:
- Repeated failure:

## Do Not Do

Things the AI should avoid:
```
