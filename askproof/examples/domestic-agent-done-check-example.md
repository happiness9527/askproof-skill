# Domestic Agent Platform Done Check Example

## Fictional Context

Tool: A fictional domestic AI agent platform

Project: CoursePilot, a course landing page generator.

## User Input

```text
国内 Agent 平台说：
已修复并重新部署，现在应该可以正常使用。
```

## AskProof Output

```text
当前判断：Done Check｜完成验收

用户真正的问题：
你想知道“已重新部署”是否等于用户真的能正常使用。

AI 当前说法是否可信：
完成可信度：低。它声称修复和部署了，但没有提供部署地址、版本、截图或用户路径验证。

已确认的信息：
- AI 声称已经修复。
- AI 声称已经重新部署。

未确认的信息：
- 部署地址或版本是什么。
- 是否验证了原来出问题的页面。
- 是否有截图、录屏、日志或错误前后对比。
- 是否确认移动端或关键入口可用。

风险点：
- “已部署”只说明可能上线了，不等于问题已经通过验收。

下一步最稳妥动作：
让 AI 提供部署和验收证据，不要只接受“应该可以”。

可直接复制给 AI 的 Prompt：
请先不要继续扩展。请提供这次修复对应的部署地址或版本、你验证过的用户路径、验证结果截图或文字记录，以及仍未验证的内容。如果只是完成了发布但没有验收，请标注“已发布但未验证”。

你现在最应该问 AI 的一句话：
请提供部署地址或版本、你验证过的用户路径、验证结果，以及哪些内容仍未验证。
```
