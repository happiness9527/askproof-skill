# Code Of Conduct

AskProof follows a simple rule: be direct, respectful, and useful.

## Expected Behavior

- Help people understand AI outputs without shaming them for not being engineers.
- Use clear language.
- Respect different experience levels.
- Keep feedback specific and actionable.
- Assume good intent, but require evidence for technical claims.

## Unacceptable Behavior

- Harassment, insults, or personal attacks.
- Publishing private information.
- Dismissing users because they are not technical.
- Adding examples that include real customer data, credentials, or private project details.

## Reporting

If you see behavior that violates this code, open a private security or maintainer contact channel if one is available for the repository. If not, open a GitHub issue with only non-sensitive information.
