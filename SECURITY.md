# Security Policy

AskProof is a local Skill project. It should not require API keys, credentials, telemetry, or network access.

## Supported Versions

| Version | Supported |
| --- | --- |
| 0.1.x | Yes |

## Reporting A Vulnerability

Please report security issues through GitHub security advisories when available.

Do not paste secrets, private logs, tokens, customer data, or `.env` values into public issues.

## Privacy Rules

- Use fictional examples only.
- Do not commit local memory containing private project details.
- Keep `docs/askproof-memory/private/` ignored.
- Keep `.env`, `*.local`, and `project-profile.local.md` ignored.
